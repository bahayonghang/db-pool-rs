## ADR 模板

使用以下模板创建新的 ADR：

```markdown
# ADR-XXX: [决策标题]

## 状态
[提案中/已接受/已弃用/已替代]

## 背景
[描述驱动这个决策的上下文和问题]

## 决策
[描述我们做出的决策]

## 后果
[描述应用这个决策的结果，包括正面和负面影响]

## 实施
[描述实施细节和步骤]

## 风险缓解
[描述相关风险和缓解措施]
```

## 决策原则

1. **简单性优先** - 选择简单可维护的方案
2. **渐进式改进** - 支持逐步升级和演进
3. **风险可控** - 每个决策都考虑风险缓解
4. **性能与可维护性平衡** - 不盲目追求极致性能
5. **开发者友好** - 降低使用和开发门槛
